# Gym Workout Tracker

A modern, responsive workout tracking application with progress tracking, achievements, and session timing.

## Features
- Multiple workout splits (3, 4, 5, 6 days)
- Progress tracking with achievements
- Session timer
- Weight logging
- Local storage for data persistence
- Responsive design
